
# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

This project is an end-to-end Data Analytics project that analyzes customer shopping behavior using Python, SQL, and Power BI. The objective is to identify purchasing patterns, customer preferences, and business insights that can help improve sales strategies and customer engagement.

The project demonstrates the complete data analytics workflow, from data cleaning and exploratory data analysis (EDA) to SQL querying and interactive dashboard creation.

---

## 🎯 Business Problem

Retail businesses collect large amounts of customer data, but transforming this data into actionable insights is often challenging.

This project aims to answer questions such as:

- Which product categories generate the highest revenue?
- How does customer age influence purchasing behavior?
- What is the average purchase amount?
- Which subscription status contributes the most revenue?
- Which shipping methods are most frequently used?
- How do customer ratings vary across different categories?

# 📈 Key Insights

- Clothing and Footwear are among the most purchased categories.

- Middle-aged customers generate a large share of purchases.
- Certain shipping methods are preferred over others.
- Average review ratings remain consistently positive across categories.
